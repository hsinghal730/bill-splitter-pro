# Bill Splitter Pro

Vite + React app you can run locally in any modern browser.

## Run Locally

**Prerequisites:** Node.js 18+ and npm.

1. Install dependencies:
   `npm install`
2. Start the dev server:
   `npm run dev`
3. Open the app in your browser:
   `http://localhost:3000`

## Build + Preview

1. Build the production bundle:
   `npm run build`
2. Preview the production build:
   `npm run preview`
